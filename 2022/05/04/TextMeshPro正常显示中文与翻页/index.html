TextMeshPro正常显示中文与翻页

2022-05-04

TextMeshPro正常显示中文

TMP也是原先第三方收购的，一般来自这种模板里没有，都需要通过包管理器来加。但是这块文档怎么找呢？这里的话，通过关键字来找不是很规范。到Unity官方手册里Packages and feature sets>按关键字排列的包（包括自带的和不自带要通过包管理器安装的）

要解决TextMeshPro显示中文的问题，实际就是解决显示TextMeshPro字体的问题。有以下两种解决方式

  • 动态加载字体（优先使用，比较简单，性能稍微差些）
  1. 下载字体，最好是开源字体，未来在项目中可以使用
  2. 在Unity projetc窗口中，在Assets下新建Fonts文件夹，导入下载好的字体（就是把下载好的字体，解压出来的.ttf拖拽进去）
  3. 创建动态字体:在Prject窗口中，选择下载好的字体，右键-Create-TextMeshPro-Font Asset创建好的动态字体，会根据你使用过的文字，自动储存到Atlas中
  • 静态字体（静态字体效率会比动态的高，适合文字较多时使用）
  1. 下载一个“常用汉字”的txt文件，必须使用Unicode编码保存，比如utf-8，将其导入Unity
  2. 在Unity菜单中，选择Window–TextMeshPro–Font Asset Creator，打开窗口
  • Source Font File，选择添加好的中文字体文件
  • Character Set：选中Characters from File 从文件获取文字
  • Character File：选中txt文件
  • Generate Font Atlas：点击按钮，生成静态字体图集（比较慢的一个过程）
  • Save：保存生成的字体asset

字体后备 Font Fallback

某些动态字体，在使用中文标点符号和一些特殊文字符号的时候，依旧会出现方块，这个时候可以使用TextMeshPro的FontFallback。Font Fallback可以从另一个FontAsset中获取不包含当前FontAsset中的字符。



TextMeshPro翻页

当Text Input内容超出文本框，如果还要显示，就会溢出。如何解决溢出呢？

  1. 选中DialogTMP找到Overflow（溢出），把它设置成Page
  2. Page就是分页显示，后面那个文本框就是显示第几页，静态的、刚开始起始状态下显示第几页

设置完以后，溢出的文字不会再显示了。那么就要添加翻页的功能，翻页需要翻页键。

我们可以通过在对话时，按下一个按钮，比如空格，激活翻页功能。

  • 在Update方法中，通过if判断是否按下空格，最好使用GetKetUp，避免按一次键，事件多次触发
  • 设定一个变量，代表当前页码，还要知道总页数
  • 每次按键，只要没到最后一页，都是当前页数加一；是最后一页，让其循环到第一页
